settings

Почему без сервера не синхронизируется “само”?
GitHub Pages отдаёт статические файлы и не может принимать ваши изменения. Поэтому по умолчанию всё хранится в вашем браузере (IndexedDB) — автосохранение без кнопок. Если хочешь, чтобы одна и та же база была и на ПК, и на телефоне — нужно писать в облако через API. Здесь это сделано через GitHub Contents API (файл в репозитории) или через Gist. Токен хранится только у тебя в браузере.

Рекомендация: создай отдельный файл crate/onoi_notes/db.json в репозитории и дай токену права repo. Если боишься токена — используй Gist и права gist.
